# 数据来源使用政策 (Source Use Policy)

本仓库是**公开**仓库。公开内容仅限研究框架、工作流、字段规范、非敏感示例和分析计划。

## 不入库（不 commit）的内容
- 受版权保护的全文 PDF / CAJ 原文
- 机构授权下载的受保护文献
- 精确的敏感物种坐标（尤其是濒危/盗猎风险高的物种，如虎豹、某些雉类）
- Zotero 本地数据库与 OneFind 索引
- 原始相机照片/视频媒体文件
- 他人未发表的私有数据

## 可入库的内容
- 文献元数据（题名、作者、年、DOI、URL、语言、摘要）—— 不构成版权侵权
- 从公开文献抽取的结构化数值（物种记录、相机工作日、RAI 等）—— 属事实数据
- 经**泛化**处理的坐标（如保留到县级、保护区质心，或对敏感物种只给保护地/县）
- 项目自产的分析脚本、图表、清洗规则
- 工作流文档与模板

## 坐标敏感性处理规则
- 濒危/盗猎高风险物种（虎、豹、雪豹、金猫、部分雉类等）：公开数据集中坐标精度降至 `county` 或 `protected_area`，精确点位只在本地版数据库 `v2_analysis_ready_local` 保留。
- `pdf_manifest.csv` 的 `legal_use_status` 字段必须标注每个全文文件的获取与使用授权状态。
- 若不确定某文献是否可抽取/公开，先记入 `04_screening/` 并在 `12_audits/` 备案，再决定。

## 全文获取边界
仅从开放获取、机构授权、作者公开附件、期刊补充材料下载。浏览器辅助脚本仅用于在**授权页面**上自动化重复点击与命名，不作为绕过访问限制的工具。
